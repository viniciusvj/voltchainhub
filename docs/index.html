Energia como bem comum,
não commodity de monopólio.

VoltchainHub é o primeiro protocolo open-source brasileiro que une ESP32-S3, OpenEMS e Polygon PoS para criar um mercado P2P de energia solar, com custo de transação inferior a $0,001 e conformidade com a ANEEL REN 1000/2021.

Sem token especulativo próprio. Prosumidor recebe em qualquer ERC-20 listado: BRZ/BRLA (R$), USDC/USDT (US$), ETH, WBTC ou outros. Swap atômico via Uniswap v3 no settlement. Fee do protocolo: 0,5% fixo.

Por que VoltchainHub?

🔓

100% Open-Source

Apache 2.0 em todas as camadas, firmware, contratos, adapters e SDKs. Zero lock-in, zero componente proprietário obrigatório.

🇧🇷

Brasil-First

Construído para a ANEEL REN 1000/2021. Opera dentro do sistema de compensação existente, sem depender de nova regulamentação.

🔗

Polygon PoS

Finalidade em ~2 segundos, custo <$0,001 por operação. LuzToken (ERC-1155) é recibo de 1 kWh verificado, não ativo especulativo. Rastreabilidade de fonte e timestamp.

📡

ESP32-S3 Edge

Medição Modbus RTU/TCP + assinatura ECDSA P-256 + MQTT over TLS. Chave privada num secure element dedicado (ATECC608B), fraude de medição impossível.

⚡

Mercado P2P Real

PowerMatcher clearing a cada 5 minutos. Preço de equilíbrio R$0,05 a R$0,15/kWh, você paga até 10x menos que a distribuidora.

💳

Pagamento Flexível

Prosumidor escolhe em que ERC-20 quer receber: BRZ/BRLA (R$), USDC/USDT (US$), ETH, WBTC, ou qualquer token com liquidez no Uniswap v3. VoltMarketplace faz o swap atomicamente. Off-ramp PIX via Transfero/Ripio.

🏛️

DAO Governance

Fase 1-2: multisig Gnosis Safe 3/5. Fase 3: Aragon OSx com token de governança dedicado. Treasury transparente 100% on-chain.
